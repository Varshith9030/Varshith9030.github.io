Mega-Finder/recon scanner

Nov 2025 · Python
async aiohttp recon misconfig javascript-analysis cloud-discovery

An async, high-concurrency reconnaissance engine that automates the boring parts of an external engagement. Crawl a target list, find what shouldn't be exposed, and dump structured output ready for triage. Ships with a 1M-entry wordlist and severity-classified output for direct intake into a report.

Detections .env leaks · .git repository exposure · backup archives (zip, sql, tar.gz, bak) · Swagger / OpenAPI definitions · S3 and GCS buckets · GraphQL introspection · CORS misconfigurations · directory listing · CI / CD panel exposure · JS secret extraction (API keys, tokens, Firebase configs) · phpinfo · debug endpoints

Architecture: domain-restricted HTML crawler + deep JavaScript analyzer + async HTTP probe layer with concurrency controls and rate limiting. Per-target output as JSON + CSV + the actual leaked artifacts written to disk.

Web App Penetration Testing Tool/vulnerability scanner

Dec 2024 · Python
selenium owasp-top-10 xss sqli csrf reportlab

Python-based web vulnerability scanner that crawls targets and tests for OWASP Top 10 issues. Uses Selenium for headless-browser detection of DOM-based and JavaScript-rendered flaws that static scanners typically miss. Outputs structured PDF reports with severity ratings and proof-of-concept payloads.

Tests Cross-Site Scripting (Reflected + DOM-based) · SQL Injection · Cross-Site Request Forgery · common OWASP Top 10 patterns
